# EDA-Student-Performance-Analysis
📊 Exploratory Data Analysis for Discovering Patterns and Insights

Overview

This project presents a comprehensive Exploratory Data Analysis (EDA) of a Student Performance dataset to identify meaningful patterns, relationships, and factors influencing academic performance. The project combines statistical analysis with interactive visualizations to derive valuable insights and recommendations.

---

🎯 Objective

The primary objective of this project is to:

- Understand the characteristics of the dataset.
- Perform data cleaning and preprocessing.
- Analyze statistical properties of the data.
- Visualize patterns and relationships between variables.
- Identify key factors affecting student performance.
- Generate insights and recommendations based on the analysis.

---

💡 Key Insights

- Study Hours show a strong positive correlation with Final Score.
- Attendance significantly impacts academic performance.
- Previous Scores are reliable indicators of future performance.
- Excessive Internet Usage shows a mild negative relationship with Final Score.
- Consistent study habits contribute to better outcomes.
